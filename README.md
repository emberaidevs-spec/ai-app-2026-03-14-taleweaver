> Tale Weaver is an AI-powered story and poem generator that helps users create unique and engaging content. With customizable prompts, tone, and style, users can generate stories and poems that reflect their personality and creativity.

## ✨ Features

- Customizable prompts
- Tone and style selection
- Shareable content
- User profiles to save and view generated content

## 🛠 Tech Stack

- Frontend: Pure HTML / CSS / JavaScript
- Backend: Vercel Serverless Functions (Node.js)
- AI: Groq API (Llama 3.3 70B)
- Groq AI API via Vercel serverless function + specific browser APIs for sharing and user profiles
- Dark theme with glassmorphism UI
- Fully responsive (mobile + desktop)
